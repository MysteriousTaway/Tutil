# TUTIL
Tutil stands for <b>T</b>away's <b>Util</b>ities`. It is a wrapper/library that I use for making my projects. If i use a piece of code often then I add it here.
## Features

- Runtime configuration
- File manager
- Posting to API
- Stopwatch
- Time formatting
- Logging
